BPM 历史链接
每个练习动作绑定独立的 BPM 记录：
用户选择 "双击 开-闭-开"
→ 查询该动作最后一次成功 BPM: 120
→ 节拍器自动设置为 120 BPM
→ 练习结束后记录新的成功 BPM
→ 下次打开自动加载 最新 BPM
存储结构：
bpm_history {
  exercise_id: string
  bpm: number
  success: boolean
  mode: 'normal'|'accel'|'silent'
  timestamp: Date
}
